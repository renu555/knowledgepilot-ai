# KnowledgePilot AI

KnowledgePilot AI is an on-premise multimodal AI assistant that enables secure search and retrieval across enterprise knowledge sources such as PDFs, Word documents, PowerPoint presentations, screenshots, scanned files, and images.

## Project Vision

Create a fully local enterprise AI assistant capable of:

- Semantic document search
- OCR on scanned PDFs
- Screenshot understanding
- Hybrid retrieval
- Persistent AI memory
- Citation-based answers
- Multi-agent architecture

## Planned Technology Stack

| Component | Technology |
|-----------|------------|
| Language | Python |
| LLM | Ollama + Qwen |
| Vector DB | ChromaDB |
| Database | SQLite |
| OCR | PaddleOCR |
| Vision | Qwen2.5-VL |
| UI | Streamlit |
| API | FastAPI |
