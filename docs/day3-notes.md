What is Ollama?
Ollama is lightweight runtime that allows us to download ,manage and run large language model locally on our computer through a simple interface and API. 
Is Ollama itself an LLM?
Ollama is not LLM. It is runtime and model management tool that downloads, loads and serves LLMs. 
Why do we need Ollama?
We need Ollama to run open-source AI models locally without depending on cloud APIs. Models available are Qwen, Gemma, Llama, Phi, DeepSeek, Mistral.
What is a model?
Model is neural network trained on massive amounts of text so that it learns language patterns. Given a prompt, it predicts the most likely next token repeatedly until it generates a complete response.  
Why did we choose Qwen 2.5 3B for your laptop instead of a larger model?
Qwen 2.5 3B provides a good balance between performance, memory usage and response quality on a laptop with 16 GB and integrated graphics.
